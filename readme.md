### Server and Client GRPC
### Simple GRPC Server 


```angular2html
Server can be started at 5001 port using following command
1.  npm i -g @nestjs/cli
2.  npm i --save @grpc/grpc-js @grpc/proto-loader
3.  npm install
4.  npm run lint -- --fix
5. npm run start:dev
```
### Run Client

```angular2html
To Run client - 
1. cd client
1. go mod tidy
2. go run pkg/cmd/main.go
```
